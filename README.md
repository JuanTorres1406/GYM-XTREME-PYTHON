# Sistema GYM-XTREME

Aplicación web desarrollada con Django enfocada en la gestión de inventario para gimnasio.

## Funcionalidades
- CRUD de productos
- Control de stock
- Gestión de registros
- Interfaz responsive

## Tecnologías utilizadas
- Python
- Django
- Bootstrap
- MySQL

## Participación
Desarrollo del módulo de inventario, creacion de productos y diseño de interfaz.
